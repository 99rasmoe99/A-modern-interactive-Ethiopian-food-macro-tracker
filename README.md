
# 🍽️ Ethiopian Feast · Meat vs Veggie Plate & Dish Battle  
A modern, interactive Ethiopian food macro tracker

## 🌍 Overview  
**Ethiopian Feast** is a beautifully designed, interactive nutrition tool that lets users build an Ethiopian plate, compare meat vs vegetarian dishes, and instantly visualize calories and macros. It’s fast, intuitive, culturally authentic, and built for anyone who wants to explore Ethiopian cuisine through a health‑focused lens.

This project powers the **Meat vs Veggie Plate Builder** and the **Dish Battle** comparison engine.

---

## ✨ Features  
### 🧺 Build Your Plate  
- Add any meat or vegetarian entrée  
- Real‑time macro totals (kcal, protein, carbs, fat)  
- Clear plate with one click  

### ⚔️ Dish Battle  
- Compare **any meat dish vs any vegetarian dish**  
- Side‑by‑side macro breakdown  
- Instant visual difference  

### 🔍 Smart Search  
- Search dishes by name (e.g., *Doro Wot*, *Misir*, *Tibs*, *Shiro*)  

### 📊 Live Macro Engine  
- Auto‑updates totals as you add/remove dishes  
- Displays macro differences between categories  

### 📥 Export  
- Export your plate for sharing or tracking
